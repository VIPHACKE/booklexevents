# BOOKLEX Holi Event Landing Page

A cinematic, animated Holi-themed landing page built for BOOKLEX.

This project showcases a modern parallax scrolling experience with layered visuals, smooth animations, and immersive UI/UX design. It is designed to function as a premium event microsite rather than a basic festival template.

---

## 🚀 Live Concept

A visually immersive landing page featuring:

- Parallax layered background
- Animated balloons and decorations
- Interactive scroll effects
- Responsive design (mobile-first)
- Smooth transitions
- Clean modern UI

---

## 🛠 Tech Stack

- HTML5
- CSS3 (Custom Animations + Responsive Design)
- JavaScript (Parallax + Scroll Effects)
- Deployed via Vercel
- Domain managed through Hostinger
